These keys are required for the admin/internal API (ADMIN_AUTH_KEY) and the lightweight remote API (AUTH_KEY). AUTH_KEY must match the AUTH_KEY set in CubicClient. They are encrypted and stored securely in data/auth_keys.bin. You must set them before using the panel.

Keep these somewhere safe. Changing them will immediately update API access.
